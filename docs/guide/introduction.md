# 介绍

Haura UI 是一个克制而精致的 Vue 3 组件库。

## 设计理念

**克制而精致，耐看而不乏味**

Haura UI 追求的是一种克制的精致感。我们不堆砌特效，而是通过微妙的光效和节奏感的动效，让界面在细节处展现品质。

### 核心特性

| 特性 | 说明 |
|------|------|
| **微妙光效系统** | 聚焦、悬停、选中时的柔和光晕 |
| **动效节奏感** | 独特的缓动曲线和时序编排 |
| **完整 Props** | 对标主流 UI 库的属性配置 |
| **6 套主题** | 亮色/暗色 × 中性/暖色/冷色 |
| **TypeScript** | 完整的类型定义 |
| **国际化** | 内置多语言支持 |

## 品牌寓意

**H + Aura = 你的光环**

Haura（光环）代表每个开发者都能在项目中展现自己的光芒。我们提供克制而精致的基础，让开发者能够专注于创造独特的体验。

## 技术栈

- **Vue 3.3+** — Composition API
- **TypeScript 5.0+** — 完整类型支持
- **UnoCSS** — 原子化 CSS
- **VitePress** — 文档站点
