# Radio 单选框

单选框组件，用于在一组选项中选择一个。

## 禁用状态


## 按钮样式

使用 `button` 属性显示为按钮样式。


## 带边框

使用 `border` 属性显示边框。


## API

### Props

| 属性 | 说明 | 类型 | 默认值 |
|------|------|------|--------|
| modelValue | 绑定值 | `string` / `number` | — |
| value | 选项值 | `string` / `number` | — |
| disabled | 是否禁用 | `boolean` | `false` |
| button | 按钮样式 | `boolean` | `false` |
| border | 显示边框 | `boolean` | `false` |

### Events

| 事件名 | 说明 |
|--------|------|
| change | 值变化时触发 |
