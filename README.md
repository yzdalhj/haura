# Haura UI

> 面向沉浸式交互的 Vue 3 组件系统

不是组件的堆砌。是界面生命体征的重构。

---

## 为什么不是"另一个组件库"

市面上的组件库解决的是"有没有"的问题。

Haura UI 解决的是"好不好用"的问题——在肉眼可见与不可见之处，都有精心调校。

- **聚焦时的光晕**，不是简单的边框变色，而是柔和的能量场扩散
- **列表悬停切换**，不是突兀跳变，而是背景条像磁铁一样滑动跟随
- **按钮点击**，不是瞬间响应，而是从触点向外扩散的涟漪反馈

这些细节单独看微不足道，组合在一起形成**品牌级的体验一致性**。

## 架构设计

### 分层包结构

```
haura-ui/
├── @haura/components    # 18 个 P0 核心组件
├── @haura/theme         # 6 套设计令牌预设
├── @haura/motion        # 动效引擎与手势系统
├── @haura/effects       # 粒子、视差等装饰层
├── @haura/utils         # SSR 工具与 DOM 工具
└── @haura/locale        # 运行时国际化
```

### 交互分层策略

| 层级 | 包体积 | 引入方式 | 说明 |
|------|--------|----------|------|
| 核心层 | ~8KB | 内置 | 涟漪、悬停指示器、状态过渡 |
| 增强层 | 按需 | Vue 指令 | v-ripple、v-magnetic、v-tilt |
| 装饰层 | ~10-20KB | 独立包 | 粒子爆发、滚动视差 |

**产品配置建议**：
- 企业后台：核心层 + 少量增强
- C 端产品：核心层 + 增强层
- 营销页面：全部启用

## 核心系统

### 1. 光效系统

不是阴影。是基于视觉层级的光子模拟。

| 光效类型 | 触发场景 | 技术实现 |
|----------|----------|----------|
| 聚焦光晕 | 输入框聚焦 | CSS 自定义属性 + 硬件加速合成 |
| 悬停微光 | 按钮/卡片悬停 | 透明度 30% 的柔和发光 |
| 选中脉冲 | 选中状态 | 短暂脉冲后保持微光 |
| 涟漪扩散 | 点击操作 | 从点击点向外扩散的圆形波纹 |
| 呼吸光效 | 加载状态 | 2-3s 周期性明暗变化 |

**参数规范**：
- 发光半径：0-8px（克制，避免刺眼）
- 过渡时长：200-300ms
- 颜色透明度：20-30%

### 2. 动效节奏系统

独特的缓动曲线形成品牌记忆。

| 曲线名称 | cubic-bezier | 手感描述 |
|----------|--------------|----------|
| Ease Out | (0, 0, 0.2, 1) | 轻盈结束 |
| Ease In Out | (0.4, 0, 0.2, 1) | 平衡自然 |
| Bounce | (0.34, 1.56, 0.64, 1) | 轻微弹跳 |
| Snappy | (0.6, 0, 0.7, 0.35) | 干脆利落 |

**时序编排**：
- 微交互：100-150ms（快速响应）
- 状态切换：200-250ms（流畅自然）
- 页面转场：300-400ms（有序展开）
- 编排动画：间隔 50ms（错落有致）

### 3. 玻璃效果系统

用于悬浮层和导航，增强层次感。

```css
.glass-effect {
  background: rgba(255, 255, 255, 0.75);
  backdrop-filter: blur(12px);
  border: 1px solid rgba(255, 255, 255, 0.1);
}
```

**使用边界**：仅用于悬浮层（导航栏、模态框、抽屉），不用于内容承载区域。

### 4. 风格预设系统

6 套开箱即用的主题预设。

| 预设 | 亮度 | 色调 | 适用场景 |
|------|------|------|----------|
| Light | 亮色 | 中性 | 通用默认 |
| Dark | 暗色 | 中性 | 开发者工具 |
| Light Warm | 亮色 | 暖色 | 社交、生活类产品 |
| Dark Warm | 暗色 | 暖色 | 夜间阅读 |
| Light Cool | 亮色 | 冷色 | 科技、金融 |
| Dark Cool | 暗色 | 冷色 | 专业后台 |

## 设计原则

**克制而精致，耐看而不乏味**

- **克制**：装饰只在关键位置出现，渐变一个界面不超过 2 处
- **留白**：充足的负空间，元素不拥挤
- **一致**：统一的设计语言贯穿始终
- **微妙**：细节需要发现，不是一眼看穿
- **功能导向**：每个设计决策服务于体验，不为炫技而动效

## 快速开始

```bash
npm install haura-ui
```

```ts
import { createApp } from 'vue'
import HauraUI from 'haura-ui'
import 'haura-ui/style.css'

createApp(App)
  .use(HauraUI, {
    theme: 'auto',        // 响应系统主题
    motion: 'natural',    // 自然动效模式
  })
  .mount('#app')
```

## 无障碍

尊重每一次用户选择。

- `prefers-reduced-motion` — 静谧模式，动效归零
- `prefers-color-scheme` — 跟随系统明暗
- WAI-ARIA 语义化标记
- 键盘导航支持
